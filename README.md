# Document Conversion Demos

This repository is organized for multiple document-to-PDF conversion implementations.

## Implementations

- `aspose_service/`: Python + FastAPI + Aspose implementation. This service converts Office and email files to PDF through a Linux-compatible HTTP API.
- `java_aspose_service/`: Java + Aspose implementation. This version runs all Aspose products in one JVM and is the simpler single-container deployment target.

Keep each implementation in its own top-level folder so dependencies, runtime files, and service code do not mix.
